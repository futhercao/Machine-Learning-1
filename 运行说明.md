# 赛道一：ModelNet40 — 运行说明

## 环境
- Python 3.10，PyTorch 2.0+ (CUDA 11.7+)
- numpy
- 单卡训练，~16 GB 显存即可（RTX A6000 实测 ~5h / 200 epoch）

## 目录约定
```
ModelNetProject/
├─ code/                    # 源代码（本目录）
│   ├─ preprocess.py
│   ├─ data_loader.py
│   ├─ models.py
│   ├─ train.py
│   ├─ predict.py
│   └─ make_submission.py
├─ preprocessed/
│   └─ shape_names.npy
├─ checkpoints/
│   └─ pointmlp.pt
└─ submit.csv
```

## 一、从零复现

### 1. 预处理
把官方 `ModelNet40_Train` 解压到 `<RAW>`（包含 `modelnet40_shape_names.txt`、`modelnet40_train.txt` 与各类别子目录），然后：
```bash
python code/preprocess.py --raw <RAW> --out preprocessed
```
得到 `data.npy / labels.npy / ids.npy / shape_names.npy`。

### 2. 训练
```bash
python code/train.py --epochs 200 --batch_size 32 --seed 2026
```
在 `checkpoints/pointmlp.pt`（best val iAcc）、`checkpoints/pointmlp_bestcacc.pt`（best val cAcc）、`checkpoints/pointmlp.pt.last`（最终 epoch）保存权重。

## 二、直接复现提交

### 1. 预测（npy 测试集）
给定 `test_data.npy`（形状 `(M, P>=1024, 6)`）与可选 `test_ids.npy`（长度 M 的样本 id 列表）：
```bash
python code/predict.py \
    --ckpt checkpoints/pointmlp.pt \
    --test_npy test_data.npy --ids_npy test_ids.npy \
    --shape_names preprocessed/shape_names.npy \
    --tta 10 --out submit.csv
```

### 2. 预测（原始 .txt 测试目录）
```bash
python code/predict.py --ckpt checkpoints/pointmlp.pt \
    --test_dir <TEST_DIR> --shape_names preprocessed/shape_names.npy --tta 10 --out submit.csv
```

### 3. 预测（id 列表 + 资源根目录）
```bash
python code/predict.py --ckpt checkpoints/pointmlp.pt \
    --test_list test.txt --test_root <RAW>/raw_resampled \
    --shape_names preprocessed/shape_names.npy --out submit.csv
```

输出 `submit.csv` 每行 `sample_id,class_name`，无表头。

### 4. 应急快速模式（CPU / 限时场景）
```bash
python code/predict.py --fast --ckpt checkpoints/pointmlp.pt \
    --test_npy test_data.npy --ids_npy test_ids.npy \
    --shape_names preprocessed/shape_names.npy --out submit.csv
```
单模无 TTA，CPU 上 ~30 秒完成。

## 三、打包提交
```bash
python code/make_submission.py --names "姓名1-学号1,姓名2-学号2,姓名3-学号3" --csv submit.csv
```
生成 `submit/赛道1-<姓名学号>.zip`，内含 code + checkpoints + submit.csv + 说明文档。
