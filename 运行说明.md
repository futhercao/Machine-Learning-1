# 赛道一：ModelNet40 — 运行说明

## 环境
- Python 3.10，PyTorch 2.0.1 (cu117) 或更高
- numpy、einops（可选）
- 单卡训练，~16–24 GB 显存即可

## 目录约定
```
ModelNetProject/
├─ code/                    # 源代码（本目录）
│   ├─ preprocess.py
│   ├─ data_loader.py
│   ├─ models.py
│   ├─ train.py
│   ├─ predict.py
│   └─ make_submission.py
├─ preprocessed/
│   └─ shape_names.npy
├─ checkpoints/
│   ├─ pointnet2_ssg_s2026.pt
│   ├─ pointnet2_ssg_s42.pt
│   ├─ pointnet2_msg_s2026.pt
│   ├─ dgcnn_s2026.pt
│   └─ dgcnn_s42.pt
└─ submit.csv
```

## 一、从零复现
### 1. 预处理
把官方 `ModelNet40_Train` 解压到 `<RAW>`（包含 `modelnet40_train.txt` 与各类别子目录），然后
```bash
python code/preprocess.py --raw <RAW> --out preprocessed
```
得到 `data.npy / labels.npy / ids.npy / shape_names.npy`。

### 2. 训练
```bash
# 主力模型
python code/train.py --model pointnet2_ssg --seed 2026 --epochs 200 --batch_size 32
python code/train.py --model pointnet2_ssg --seed 42   --epochs 200 --batch_size 32
python code/train.py --model pointnet2_msg --seed 2026 --epochs 200 --batch_size 24

# 增加集成多样性
python code/train.py --model dgcnn --seed 2026 --epochs 200 --batch_size 24
python code/train.py --model dgcnn --seed 42   --epochs 200 --batch_size 24
```
每个模型在 `checkpoints/<model>_s<seed>.pt` 留下 best val ckpt。

## 二、直接复现提交
### 1. 预测（npy 测试集）
若给定 `test_data.npy`（形状 `(M, P>=1024, 6)`）与可选 `test_ids.npy`（长度 M 的样本 id 列表）：
```bash
python code/predict.py \
    --ckpts checkpoints/pointnet2_ssg_s2026.pt \
            checkpoints/pointnet2_ssg_s42.pt \
            checkpoints/pointnet2_msg_s2026.pt \
            checkpoints/dgcnn_s2026.pt \
            checkpoints/dgcnn_s42.pt \
    --test_npy test_data.npy --ids_npy test_ids.npy \
    --shape_names preprocessed/shape_names.npy \
    --tta 10 --out submit.csv
```

### 2. 预测（原始 .txt 测试目录）
若给定一个测试根目录（含每个类别子目录或扁平 `{id}.txt`）：
```bash
python code/predict.py --ckpts ... --test_dir <TEST_DIR> --shape_names preprocessed/shape_names.npy --tta 10 --out submit.csv
```

### 3. 预测（id 列表 + 资源根目录）
```bash
python code/predict.py --ckpts ... --test_list test.txt --test_root <RAW>/raw_resampled --shape_names preprocessed/shape_names.npy --out submit.csv
```

输出 `submit.csv` 每行 `sample_id,class_name`，无表头。

## 三、打包提交
```bash
python code/make_submission.py --name 姓名 --sid 学号 --csv submit.csv
```
生成 `submit/<姓名>_<学号>.zip`，内含 code + checkpoints + submit.csv + 两份说明文档。
