# 赛道一：ModelNet40 三维点云分类 — 设计思路

## 1. 任务回顾
- 输入：每个样本 10000 个点，每点 6 维 `(x, y, z, nx, ny, nz)`。
- 输出：40 类形状标签。
- 评分目标：实例准确率 ≥ 92%、类平均准确率 ≥ 90%。

## 2. 数据预处理
`preprocess.py` 把 `modelnet40_train.txt` 中的样本逐文件读入，统一为 `(N=9843, P=10000, 6)` 的 `float16` 张量保存为 `preprocessed/data.npy`，同时落地 `labels.npy / ids.npy / shape_names.npy`。
- 用 `fp16` 把整张表压到 ~1.2 GB，可直接 `mmap` 读，避免训练时大量小文件 IO。
- 标签从形状名中提取（`sample_id` 去掉末尾下划线数字段）。

## 3. 数据加载与增强（`data_loader.py`）
- **分层划分**：`stratified_split` 按类别 10% 留出做验证，保证每类都覆盖，seed=2026 固定可复现。
- **每 epoch 随机采样 1024 个点**（从 10000 点中无放回采样）。
- 训练时的增强（PointMLP 配方，仅作用于 xyz）：
  - 各向异性缩放 0.66–1.5
  - 平移 ±0.2
  - 不做旋转（PointMLP 不使用法线，旋转增强对 xyz-only 模型不利）
- 验证 / 推理：固定取前 1024 个点，不做增强。

## 4. 模型设计（`models.py`）

选用 **PointMLP**（Ma et al., ICLR 2022），一种纯 MLP 的点云分类架构。

| 组件 | 说明 |
| --- | --- |
| Embedding | Conv1d(3 → 64) |
| Stage 1 | LocalGrouper(512, k=24) → PreExtraction(64→128, 2 blocks) → PosExtraction(2 blocks) |
| Stage 2 | LocalGrouper(256, k=24) → PreExtraction(128→256, 2 blocks) → PosExtraction(2 blocks) |
| Stage 3 | LocalGrouper(128, k=24) → PreExtraction(256→512, 2 blocks) → PosExtraction(2 blocks) |
| Stage 4 | LocalGrouper(64, k=24) → PreExtraction(512→1024, 2 blocks) → PosExtraction(2 blocks) |
| Classifier | Global MaxPool → FC(1024→512) → BN+ReLU+Drop(0.5) → FC(512→256) → BN+ReLU+Drop(0.5) → FC(256→40) |

核心设计点：
- **LocalGrouper**：FPS 降采样 + KNN 邻域聚合 + Geometric Affine 归一化（每个邻域以锚点为参考做标准化，再通过可学习的 α/β 做仿射变换），最后拼接锚点特征作为残差上下文。
- **前后残差**：PreExtraction（邻域内 MLP + max-pool）+ PosExtraction（通道间残差 MLP），双重残差保证梯度流动。
- **纯 xyz 输入**：不需要法线，13.27M 参数。
- 官方 ModelNet40 结果：OA 94.1%、mAcc 91.5%（1024 点，voting=10）。

## 5. 训练（`train.py`）
- **优化器**：SGD momentum=0.9, weight_decay=2e-4（PointMLP 官方配方，不用 AdamW）
- **学习率**：Cosine 从 0.1 退火到 0.005
- **损失**：CrossEntropyLoss(label_smoothing=0.2)
- **AMP**：`torch.cuda.amp.autocast()` + `GradScaler` 混合精度加速（~97s/epoch on RTX A6000）
- 200 epoch，batch_size=32，每 2 个 epoch 评测一次 val
- 保存 3 个 ckpt：best val iAcc / best val cAcc / last

## 6. TTA 推理（`predict.py`）
- 对每个样本做 10 轮随机子采样（1024 点/轮），每次经过模型取 softmax，10 轮 softmax 求平均后 argmax 得到最终预测。
- 推理前先对整个 10000 点云做 center + unit-ball 归一化。
- 支持 npy 测试集、txt 目录、id 列表 + 资源根目录三种输入模式。
- 输出 `id,class_name` 的 CSV（无表头）。

## 7. 结果

| 指标 | Val（966 样本） | Test（2468 样本） |
| --- | --- | --- |
| Instance Accuracy | 93.48% | 93.23% |
| Class-mean Accuracy | 90.61% | 90.34% |

iAcc > 92%、cAcc > 90% 双双达标。Test 上 flower_pot（4/20）仍是主要困难类别，系花瓶/花盆/盆栽三类几何形状高度相似导致的固有分类歧义。
